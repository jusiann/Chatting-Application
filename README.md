## 🇹🇷 Türkçe

### Proje Hakkında
Chatting Application, kurumsal iletişimi modernize eden, güvenli ve kullanıcı dostu bir mesajlaşma platformudur. Web, mobil ve masaüstü platformlarında sorunsuz çalışan uygulamamız, iş dünyasının iletişim ihtiyaçlarını tek bir noktada çözer.

### Özellikler
- 🔐 Güvenli kimlik doğrulama sistemi
- 👥 Departman ve grup yönetimi
- 💬 Gerçek zamanlı mesajlaşma
- 📱 Cross-platform destek (Web, iOS, Android)
- 📂 Dosya paylaşımı
- 🔍 Gelişmiş arama özellikleri
- 👤 Kişisel profil yönetimi

### Teknolojiler
- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Mobile:** Flutter
- **Veritabanı:** MongoDB
- **Gerçek Zamanlı İletişim:** Socket.io

### Kurulum
```bash
# Repository'yi klonlayın
git clone https://github.com/your-username/Chatting-Application.git

# Backend için
cd backend
npm install
npm start

# Frontend için
cd frontend
npm install
npm run dev

# Mobile için
cd mobile
flutter pub get
flutter run
```

## 🇬🇧 English

### About
Chatting Application is a secure and user-friendly messaging platform that modernizes corporate communication. Our application, which works seamlessly on web, mobile, and desktop platforms, solves the communication needs of the business world in one place.

### Features
- 🔐 Secure authentication system
- 👥 Department and group management
- 💬 Real-time messaging
- 📱 Cross-platform support (Web, iOS, Android)
- 📂 File sharing
- 🔍 Advanced search capabilities
- 👤 Personal profile management

### Technologies
- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Mobile:** Flutter
- **Database:** MongoDB
- **Real-time Communication:** Socket.io

### Installation
```bash
# Clone the repository
git clone https://github.com/your-username/Chatting-Application.git

# For Backend
cd backend
npm install
npm start

# For Frontend
cd frontend
npm install
npm run dev

# For Mobile
cd mobile
flutter pub get
flutter run
```
